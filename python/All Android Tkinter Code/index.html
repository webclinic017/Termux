Android tkinter Demos

This page collects Android screenshots of all the tkinter-GUI programs described in this doc. Shots are grouped by program, and can be viewed in slideshow mode by tapping Auto on any image-viewer page. Among the subfolders here, CODE hosts all the patch files needed to run these programs on Android, and Sizing has additional screenshots that demo the effects of DPI and font settings. Some shots here also demo issues that were resolved by work-arounds described in the tkinter or related Mergeall docs.

Subfolders here:

  • CODE
  • Sizing
  • etc
