Android tkinter Sizing Demos

This page collects Android screenshots that capture the effects of program font configurations and Pydroid 3 DPI settings on tkinter GUIs like those described here. Both fonts and DPI can be tailored to better size GUIs to screens, as described in more detail by the sidebar here. For those keeping score, these shots were all taken on a Galaxy Note20 Ultra in 2020. For more Android tkinter demos, see this folder's parent.
