[+] Installation :-
  
    cd /sdcard/github/Termux/python/
    git clone https://github.com/TechnicalMujeeb/Termux-Lazyscript.git
    mv -v Termux-Lazyscript Lazyscript
    cd Lazyscript
    chmod +x *
    sh setup.sh
    python3 ls.py
    rm -rf .git/

[+] usage :-

    python ls.py

(here simply type number to use that tool)
