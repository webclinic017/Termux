Run pickcolor.py for a simple #RRGGBB color selector, for use in your
config files.

Run fixeoln.py and fixeoln-all.py to convert text files in this package 
to Unix or DOS end-line format, if the files look odd in your viewers:

    ~/../docetc/Tools$  python3 fixeoln-all.py tounix ../.. utf8    # Unix
    C:\..\docetc\Tools> py -3   fixeoln-all.py todos  ..\.. utf8    # Windows

Run ls.py for a simple recursive tree content listing (a quick look
at your folders and subfolders).
